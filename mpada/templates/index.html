Kosovo Assembly MP Asset Declaration API

LAW No. 04/L-050 "defines obligations of senior public officials to declare their property, revenues and the origin and obligation of Agency to control declared property and origin of property as well as obligations of all public senior officials to declare gifts and the origin of gifts."

This Application Programming Interface (API) gives software developers access to the Asset Declrations of the Kosovo Assembly's MPs. This data is compiled by Democracy for Action (D4D).

We care about sharing and improvements so the software behind this API is free and open under the GNU GPL v2.0 license. Anyone who distributes the code or a derivative work is required to make the source available under the same terms.


Disclaimer

This API is an alpha version. This means that it is still under development and may be subject to bugs. You can help by contacting us to report issues or contribute fixes and improvements to the code base.


Getting Data

Retrieving data from the API is done through RESTful HTTP GET Requests.


MPs Grouped By Parties

Get all MPs and group them by the Parties they are affiliated with:
/aggregate/mps-grouped-by-parties


Declaration Years

Get all the years in which a Party declared its assets:
/aggregate/years-mps-declared/<string:party_slug>

Get all the years in which a specific Party member declared his assets:
/aggregate/years-mps-declared/<string:party_slug>/<string:mp_slug>

Get all the years in which all Parties declared their assets:
/aggregate/years-parties-declared


Asset Declarations

Get the asset declarations for all the MPs in a given Party:
/<string:party_slug>

Get the asset declarations for all MPs given Party for a specific year:
/<int:year>/<string:party_slug>

Get the asset declarations for a specific MP in a given Party:
/<string:party_slug>/<string:mp_slug>

Get the asset declarations for a specific MP in a given Party for a specfied year:
/<int:year>/<string:party_slug>/<string:mp_slug>


Median Asset Declarations

Get all median asset declarations through the years:
/aggregate/median

Get the median asset declarations for a specific year:
/aggregate/median/<int:year>

Get the median asset declarations for a specific Party:
/aggregate/median/<string:party_slug>

Get the median asset declarations: for a specific Pary in a given year:
/aggregate/median/<int:year>/<string:party_slug>


Aggregate Sum Asset Declarations

Get the sum of all asset declarations through the years:
/aggregate/sum

Get the sum of all asset declarations through the years for a specified party:
/aggregate/sum/<string:party_slug>

Get the sum of all asset declarations through the years for a specified MP:
/aggregate/sum/<string:party_slug>/<string:mp_slug>

Sample Requests

Get the years where MP Albin Kurti from VV declared his assets:
{{ config.BASE_PATH }}/aggregate/years-mps-declared/vv/albin-kurti


Get the 2104 asset declarations of all MPs who are members of VV:
{{ config.BASE_PATH }}/2014/vv


Get the yearly asset declarations of VV Party member Albin Kurti:
{{ config.BASE_PATH }}/vv/albin-kurti


Get the yearly aggregate sum of VV's members asset declarations:
{{ config.BASE_PATH }}/aggregate/sum/vv
